Pi VIP Rideshare - Native Mobile Apps Package
===============================================

This folder contains everything you need to build native iOS and Android apps:

FILES INCLUDED:
1. pi-vip-rider-complete.tar.gz (148KB)
   - Rider app with blue theme
   - App ID: com.pivip.rideshare.rider

2. pi-vip-driver-complete.tar.gz (131KB)
   - Driver app with green theme
   - App ID: com.pivip.rideshare.driver

3. NATIVE_APPS_SETUP_GUIDE.md (7.8KB)
   - Complete setup instructions
   - iOS and Android build steps

QUICK START:
1. Extract both .tar.gz files
2. Follow instructions in NATIVE_APPS_SETUP_GUIDE.md
3. Build with Xcode (iOS) or Android Studio (Android)

For questions, refer to the setup guide!
